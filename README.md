# AI Agents Learning Hub

A single-page interactive learning portal for Google's **5-Day AI Agents: Intensive Vibe Coding Course**. It organizes the entire curriculum into one expandable roadmap — Course Overview, Days 1–5, and the Capstone Project — with every podcast, whitepaper, codelab, and livestream linked to its official source.

🔗 **Live site:** https://oii-nasif.github.io/5-Day-AI-Agents-Intensive-Vibe-Coding-Course-With-Google/

---

## ✨ Features

- **Interactive roadmap** — click any step to expand its topics, content chips, and resources.
- **Course Overview** — what the course is, who it's for, prerequisites, and how the intensive works.
- **Five daily units (Days 1–5)** — each with a description, "what you'll learn" topics, and tab-filtered resources.
- **Capstone Project** — the final step with all four project tracks.
- **Direct links** — every resource opens its real Kaggle whitepaper, YouTube podcast/livestream, or Google codelab.
- **Content-aware chips** — each day shows its mix at a glance (podcast · whitepaper · codelabs · livestream · estimated time).
- **Light & dark mode** — toggle in the top right; preference is remembered.
- **Fully responsive** and **self-contained** — a single HTML file with no build step or dependencies.

## 🚀 Usage

The site is a single self-contained `index.html` — no install, no build.

- **View locally:** open `index.html` in any modern browser.
- **Host on GitHub Pages:** put `index.html` at the repo root (or `/docs`) and enable Pages in **Settings → Pages**.
